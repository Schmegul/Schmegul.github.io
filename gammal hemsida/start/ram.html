Datorns RAM

Förkortningen RAM står för Random Access Memory och funkar ungefär som en vanlig CPU. Ett RAM minne är en extremt snabb processor som datorn använder för att lagra temporär data från program. RAM minnet skiljer sig från CPU:n ännu mer i det faktum att den tar bort alla data som var på den när den inte längre har ström.

Det här är en av de få komponenterna som en kontorsdator faktiskt skulle vilja ha en ganska bra version av, så att inte tid går förlorad i långa uppstartar av program och etc. Det är generellt bra att ha sitt RAM minne så bra som resten av datorn tillåter, eftersom nästan alla program på datorn blir snabbare av den.
