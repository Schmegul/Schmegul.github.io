Datorns CPU

CPU står för Central Processing Unit, men kallas ofta för bara processor. Det är den som "ger order" till alla andra komponenter om vad de ska göra enligt vad programmen som används på datorn säger till CPU:n.

CPU:er jobbar olika snabbt beroende på hur många cores som de har. En core i en CPU är som en liten CPU, så desto fler cores en CPU har, desto snabbare kan den processera data från program. Vilken CPU man köper beror på vad man tänker använda datorn till; en valig kontorsdator kräver inte en stark CPU, medan en dator som ska kunna använda CAD, skapa och ändra filmer eller spela spel kommer behöva en CPU med fler cores.
